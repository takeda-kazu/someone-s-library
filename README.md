# 📚 〇〇本棚 - Library Watch

Apple Liquid Retina XDRデザインインスパイアの個人本棚管理システム

## 🎨 デザインの特徴

### Apple風デザインシステム

このアプリケーションは、AppleのLiquid Retina XDRディスプレイの美学からインスピレーションを得た、モダンで洗練されたUIデザインを採用しています。

#### 主要なデザイン要素

1. **Liquidグラデーション**
   - 流れるような美しいグラデーション配色
   - 紫・ピンク・ブルーのアクセントカラー
   - 背景に動的なグラデーションアニメーション

2. **ガラスモーフィズム（Glassmorphism）**
   - フロストガラス効果のbackdrop-filter
   - 半透明な要素による奥行き感
   - 洗練されたレイヤー構造

3. **流麗なアニメーション**
   - スムーズなトランジション効果
   - ホバー時の滑らかな変化
   - ページ遷移のフェードイン/スライドアニメーション

4. **タイポグラフィ**
   - SF Pro Display/Textフォントファミリー
   - 大胆で読みやすい文字サイズ
   - 適切なレタースペーシング

5. **ダークモード優先**
   - 深い黒を基調とした配色
   - 目に優しいコントラスト
   - 発光するアクセントカラー

6. **レスポンシブデザイン**
   - モバイルファーストのアプローチ
   - あらゆる画面サイズに対応
   - タッチフレンドリーなインタラクション

## 🚀 主な機能

- 📖 **本の一覧表示** - グリッドレイアウトで視覚的に
- 🔍 **検索・フィルター** - タイトル、著者での絞り込み
- 📝 **詳細表示** - レビュー、インサイト、キーワード
- ✏️ **編集機能**（管理モード） - 本の追加・編集・削除
- 🤖 **AIプロンプト生成** - 読書記録をAIで活用
- 🔐 **認証システム** - Firebase Authentication連携

## 🛠️ 技術スタック

- **HTML5** - セマンティックマークアップ
- **CSS3** - カスタムプロパティ、Grid、Flexbox
- **Vanilla JavaScript** - フレームワーク不使用
- **Firebase** - Authentication & Firestore

## 🌟 デザインのポイント

### 1. 視覚的階層
- 大きなタイトル（3rem）
- 明確なセクション分け
- 適切な余白（スペーシングシステム）

### 2. インタラクション
- ホバー時の浮き上がり効果
- クリック時のスケールダウン
- スムーズなページ遷移

### 3. ガラスエフェクト
- backdrop-filter: blur(16px)
- 半透明背景
- 微細なボーダー

### 4. アニメーション
- 20秒周期の背景グラデーション移動
- 0.3秒のトランジション
- イージング関数による滑らかな動き

## 🔧 カスタマイズ

CSS変数を変更することで、簡単にデザインをカスタマイズできます：

```css
:root {
    /* 独自のカラーを設定 */
    --gradient-primary: linear-gradient(135deg, #your-color1, #your-color2);
    
    /* スペーシングを調整 */
    --spacing-lg: 2.5rem;
    
    /* フォントサイズを変更 */
    --font-size-3xl: 3.5rem;
}
```

1. **Clarity（明確さ）** - 機能が一目で分かるUI
2. **Deference（控えめさ）** - コンテンツが主役
3. **Depth（奥行き）** - レイヤーで視覚的階層を表現
